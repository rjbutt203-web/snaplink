üîß Method 1: Deploy on Vercel (Recommended)

Step 1: Create GitHub Repository

  1. Go to github.com/new
  2. Repository name: snaplink
  3. Description: "The Fastest URL Shortener"
  4. Public repository
  5. Click "Create repository"

Step 2: Push Code to GitHub

git remote add origin https://github.com/YOUR_USERNAME/snaplink.git git branch -M main git push -u origin main

Step 3: Deploy on Vercel

  1. Go to vercel.com
  2. Click "New Project"
  3. Select your GitHub repository
  4. Framework: Select "Other"
  5. Click "Deploy"
  6. Done! ‚úÖ Your app is live

üîß Method 3: Deploy on GitHub Pages
